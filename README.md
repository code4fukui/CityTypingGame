# CityTypingGame

A game where you quickly type the names of 20 randomly selected local government entities.

## Demo
https://code4fukui.github.io/CityTypingGame/

## Features
- Randomly presents 20 local government entity names to be quickly typed
- Supports keyboard input as well as clicking/Enter for progression
- Ability to retry the game by pressing Escape

## Data
- City names are sourced from the [address-japan](https://github.com/code4fukui/address-japan) dataset

## Libraries
- Uses the [HiraganaParser](https://github.com/2ndPINEW/HiraganaParser) library by [@2ndPINEW](https://twitter.com/2ndPINEW)
