# CityTypingGame

地方公共団体の名称を素早く入力するゲームです。

## デモ
https://code4fukui.github.io/CityTypingGame/

## 機能
- ランダムに20の地方公共団体名を表示し、迅速に入力する
- キーボードの入力に加え、クリックやEnterでの進行も可能
- Escキーを押してリトライできる

## データ
- 市町村名は[address-japan](https://github.com/code4fukui/address-japan)データセットから取得

## ライブラリ
- [HiraganaParser](https://github.com/2ndPINEW/HiraganaParser)ライブラリを使用

## 派生元
